# Financial Intelligence Dashboard

A premium, institutional-style financial file intelligence dashboard for CSV, XLSX, and PDF review.

## Features

- Drag-and-drop upload and file picker
- CSV and XLSX parsing in the browser
- PDF-safe placeholder messaging for production OCR/table extraction
- Smart clarification controls
- Executive KPI dashboard
- Searchable, sortable, filterable transactions table
- Transaction drill-down
- Dark/light mode
- KPI PDF export
- Spreadsheet CSV export
- Session-first privacy messaging

## Deployment

This repo is static and can be hosted directly with GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any static host.
